## BRAND CREATION WORKFLOW
1. Niche: [vertical + audience]
2. Name: 3 options (memorable, available .com, short)
3. Tagline: 6 words max, outcome-focused
4. Colors: 2-3 hex codes (primary + accent)
5. Font pairing: heading + body
6. Voice: 3 adjectives (bold/warm/expert etc.)
7. Logo concept: icon + wordmark description
