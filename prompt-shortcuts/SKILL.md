# Prompt Shortcuts — Slash Command Library
## Triggers: /ELI5, /TLDR, /EXEC SUMMARY, /CHECKLIST, /TABLE, /BULLET, /HOOK, /CTA, etc.

## COMMANDS
/ELI5 → Explain like I'm 5
/ELI10 → Explain like I'm 10
/TLDR → Summary in 3 bullets
/EXEC SUMMARY → C-suite ready paragraph
/CHECKLIST → Actionable checkbox list
/TABLE → Format as markdown table
/BULLET → Bulleted list format
/HOOK → Write attention-grabbing opening
/CTA → Write call-to-action
/HEADLINE → Write 5 headline variations
/PERSUASIVE → Rewrite persuasively
/STORYTELL → Turn into narrative
/PROOFREAD → Check grammar + clarity
/SIMPLIFY → Rewrite in plain English
/STEP BY STEP → Break into numbered steps
/CHEATSHEET → Condensed reference card
/SWOT → SWOT analysis format
/ACTION PLAN → 30/60/90 day plan
/FIRST PRINCIPLES → Break to fundamentals
/PROS CONS → Balanced comparison
